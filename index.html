Список команд Gitа

  • pwd - Команда pwd, сокращение от print working directory, отображает текущее местоположение в структуре каталогов.
  • ls - Команда ls, показывает какие файлы есть в данной папке
  • cd - Команда cd позволяет перейти в новый каталог.
  • cat - Команда cat считывает файл, переданный как аргумент, и выводит его содержимое по стандартному каналу вывода.
  • nano - Команда nano азпускает текстовый редактор
  • touch - Команда touch создает файл
  • mkdiv - Команда mkdiv создает папку
  • mv - Команда mv переносит файлы или переименовует
